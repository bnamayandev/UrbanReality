ONE REMAINING STEP — create your .env

The .env.example already has the correct values for Ollama + nemotron-3-super.
Just copy it. No edits needed unless you want to change the DB password.


THEN START THE SERVER
----------------------
  cd ~/UrbanForge/backend
  source ~/venv/bin/activate
  pip install xgboost  (first time only — was missing from earlier install)
  uvicorn main:app --host 0.0.0.0 --port 8001 --reload

Verify it's up:
  curl http://localhost:8001/health

You should see: {"status": "ok", "spatial_layers": {...}}


MAKE SURE OLLAMA IS RUNNING
-----------------------------
Nemotron must be serving before the backend can run impact analysis.
Check: curl -s http://localhost:11434/v1/models

If empty:
  tmux new -s nemotron
  ollama serve
  Ctrl+B then D to detach
